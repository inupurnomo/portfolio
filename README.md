## Installation

First, run install the dependencies:

```bash
npm install
# or
yarn
```

Then run with:

```bash
npm run dev
# or
yarn dev
```

## Building and Running for Production

```bash
npm run build
# or
yarn build
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
